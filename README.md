# Task 2 — Number Guessing Game

## Description
A GUI-based number guessing game built with Java Swing where the computer generates a random number and the user tries to guess it with higher/lower hints.

## Tech Stack
- Java Swing (GUI)
- No external libraries required

## Prerequisites
- JDK 17 or 21 — download from [https://adoptium.net](https://adoptium.net)

## How to Run

**Step 1: Navigate to the source folder**
```bash
cd task2-guessing-game/src
```
**Step 2: Compile**
```bash
javac NumberGuessingGame.java
```
**Step 3: Run**
```bash
java NumberGuessingGame
```

## Features
- Three difficulty levels — Easy (1–50), Medium (1–100), Hard (1–200)
- Hints after every guess — Too High / Too Low / Correct
- Attempt counter visible throughout the game
- Play Again option after every round
- Score tracking across multiple rounds
- Round history log panel
